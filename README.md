##  FLcasher

簡易的なキャッシャーアプリ。
adminモードで起動した端末をサーバーとして同一ネットワーク上の他の端末からアクセスし、同アプリのキャッシャーモードとキッチンディスプレイモードで連携する事が可能です。

##  使い方

1. **Admin Mode** 
アプリをadminモードで起動し、StartServerを選択する事でサーバーを起動します。（このときに表示されるIPアドレスに他のデバイスからアクセスします。）Manage Productを選択する事で、販売する商品の追加と編集が可能です。Sales Reportでは売り上げの管理、OrderHistoryでは過去の注文履歴の表示、削除が可能です。

2. **Cashier Mode**
同一ネットワーク上の他のデバイスで同アプリを起動し、キャッシャーモードを選択、レジの準備金額を入力後、admin画面に表示されるIPアドレス（ポート番号は入力不要です）を入力することでキャッシャーを開始します。販売はAdminで追加した商品のカードを選択して会計を押す事で行います。また、この画面から割引カードの追加・編集が可能です。左上の設定ボタンから、レシート印刷のオプションを表示できます。レシートの印刷には"LayoutEditorForSunmiV2"(私のgithubにてリリースしています)をインストールしたSunmiV2シリーズを使用します。Bluetoothで端末とペアリングを行い、LayoutEditorで任意のレイアウトのactiveボタンで待機します。キャッシャーモードのオプションからペアリングした印刷端末を選択し、レシート印刷をオンにすることで、会計時にレシートが印刷されます。

3. **Kitchen Display Mode** 
キッチンディスプレイモードはadmin画面に表示されるIPアドレスを入力して開始します。キャッシャーモードで会計が行われると、注文内容がアラートとともに即座に表示されます。提供時には注文を選択する事で完了状態へと変更できます。
